Private Digital Currency Based on Monero

Masari is a private, secure, and fungible cryptocurrency that focuses on usability and scalability. All Masari transactions are private by default. The sender, receiver, and amounts transferred are hidden from the public, making all transactions untraceable and unlinkable. These privacy features make Masari truely fungibile like cash or gold in real life. Masari has all of the privacy faetures of Monero but aims towards becoming a more user-friendly and scalable currency

Roadmap

Q1 2018 Subaddresses This Monero upstream feature allows a single Masari private key to issue multiple subaddresses, expanding beyond payment IDs.
Q1 2018 Multisig This Monero upstream feature will allow for trustless off-chain transactions that would otherwise require a counterparty.
Q2 2018 Uncle Mining (tentative) This feature unique to Masari allows for a more secure network and for safely reducing block emission rates for better network latency.
Q3 2018 Mobile Wallet This will expand on the GUI wallet, encompassing mobile devices for day-to-day usability.
